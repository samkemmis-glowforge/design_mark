# Glowforge exemplars — observed design language

These notes are transcribed from three glowforge.com marketing sections supplied as
reference (the source screenshots weren't available as files — drop the actual images
into this folder if you want the image-generation path to condition on them visually).

## 1. "Powerful tools for every creator" — audience cards
- Centered bold headline (lowercase-feeling geometric sans, near-black `#12151A`) + one-line
  subhead in regular weight.
- Row of 4 **rounded photo cards** (~16–24px radius), equal width, generous gutters.
- Each card: full-bleed lifestyle photo of real makers, with a **white label** lower-left
  ("For Educators", etc.) and a **teal pill CTA** ("Learn more →") below it.
- Light grey page background (`~#D8DADC`), lots of whitespace.

## 2. "Precision power" — vertical-tab feature section
- Left column: stacked **pill/rounded list items**; the active one is a **teal solid pill**
  with white text + arrow, the rest are light outlined chips.
- Center: square product photo (rounded corners). Right: heading + short body paragraph.
- Same light-grey canvas; calm, lots of breathing room.

## 3. "Made for making" — icon feature row
- Centered headline + subhead, then a **6-up row of thin-line teal icons**, each with a
  short 1–3 word caption beneath in near-black.
- Icons are single-weight line art in the brand teal.

## Recurring brand cues
- **Teal** is the action/identity color — used for CTA pills, active states, and line icons.
  The teal on the live CTAs reads as the deeper logo teal (`#16A0B0`), not the brighter
  `#26B8CE` deck swatch.
- Rounded geometric sans headlines; lowercase or sentence case; near-black body text.
- Real, warm, diverse maker photography; light-grey/neutral section backgrounds.
- Pill-shaped CTAs with a trailing `→`. Rounded cards. Airy spacing.
